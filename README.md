# Tester Supreme

Tester Supreme is an s&box game project focused on building and testing calibration spaces, modular architecture, interactive props, characters, and gameplay systems.

## Project Layout

- `TesterSupreme/Game/` - s&box-ready assets, code, scenes, prefabs, and libraries.
- `TesterSupreme/Production/` - Blender files, references, source textures, bakes, exports, and archive folders.
- `TesterSupreme/Documentation/` - pipeline, naming, material, import, and test documentation.
- `TesterSupreme/ReferenceScreenshots/` - visual references and implementation screenshots.
